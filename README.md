## :computer: Projeto

Este projeto tem o intuito de cadastrar Orfanatos informando dados como, por exemplo, nome, foto(s), instruções, descrição, horário de atendimento e se funciona nos fins de semana.

## :zap: Tecnologias

* [React.js](https://pt-br.reactjs.org)
* [React Native](https://reactnative.dev)
* [Expo](https://expo.dev)
* [TypeScript](https://www.typescriptlang.org)
* [Node.js](https://nodejs.org)

## :rocket: Como executar

1) Clone o repositório
  
2) Instale com o **yarn** as dependências do back-end (na pasta *backend*) e front-end (na pasta *web* ou *mobile*)

```shell
yarn install
```

3) Iniciar o servidor
   1) Backend
 
   ```
    yarn dev
   ```

   A aplicação estará sendo executada em `localhost:3333` .

   2) Web
   
   ```
    yarn start
   ```

   A aplicação estará sendo executada em `localhost:3000` .

   3) Mobile
   
   ```
    expo start
   ```

   **Obs:** *Para executar o app, será necessário baixar o aplicativo **Expo Go**. Após baixar, basta ler o QRCode.*
